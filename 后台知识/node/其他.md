### 机器信息
```js
const os = require('os')
os.cpus()  // 查询cpu信息
os.totalmem()  // 查询内存信息
```


### path
```js
const path = require('path')
path.extname('c:/a/b/c/d.txt');  // '.txt', 获取文件的拓展名
```


### 开发环境 和 生产环境
1. 运行之前先执行`export NDOE_ENV=production`;
2. 执行 js文件, 如`node first.js`;
```js
var port = 3000;
if (process.env.NODE_ENV === "production")
    port = 80;
```
