1. 
```
父元素包裹子元素;

.box{
    width:80%;
    height: 100vh;
    border: 1px solid;
    margin: 0 auto;
    
    /* 绝对定位 */
    position: relative;
    /* end */
}
.block{
    width: 100px;
    background-color: cadetblue;

    /* 相对定位 */
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
    /* end */
}
```

2. 
```
.parent{
    display: flex;
    justify-content: 'center';
    align-items: 'center';
}

# 横向 #
* flex-start: 头部对齐;
* flex-end: 尾部对齐;
* center: 居中;
* space-between: 两端对齐;
* space-around: 均分对齐;

# 纵向 #
* flex-start/flex-end/center ;
* stretch: 两端拉开;
```

### vmin 和 vmax
`vw`视口宽度;
`vh`视口高度;
`vmax` 取 vw 和 vh 中较大的那个;
`vmin` 取 vw 和 vh 中较小的那个;

### 相对视口居中
```
/* box */
.child{
    width: 100px;
    height: 50px;
    border: 1px solid;

    /* 居中 */
    margin: 50vh auto 0;
    transform: translateY(-50%)
}
```
